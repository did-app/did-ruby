# Sinatra app using DID Authentication

To start this example app.

1. Install dependencies `bundle install`
2. Set up an App at [did.app](https://did.app) to get your `client_id` and `client_secret`.
3. Start sinatra app with your app credentials.

```
CLIENT_ID=[client_id] CLIENT_SECRET=[client_secret] ruby web.rb
```

4. Visit [localhost:3000](http://localhost:3000).
